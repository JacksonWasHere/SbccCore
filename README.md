# SbccCore
Provides simple functions that make Java a *whole* lot easier to use:

```java
println("Hello")
```
rather than 
```java
System.out.println("Hell-o");
```
Other static functions:
```java
print()
printf()
readLine()
readFile()
readFileAsLines()
writeFile()
writeFileAsLines()
parseInt/Long/Float/Double/Boolean
```
